“通用”的来源——模板方法模式

模板方法模式（Template Method Pattern）定义了算法的骨架，并将一些步骤的具体实现延迟到子类中，以允许子类在不改变算法结构的情况下重新定义算法的某些步骤。
在 Spring Boot 中集成组件有一个通用过程，其中每一个步骤由用户自己实现，这就是模板方法模式的应用。

“通用"的骨架—— IoC 容器

  • IoC 容器提供了模板方法模式中“算法的骨架”
  • 控制反转是 Spring Boot 集成其它组件的核心
